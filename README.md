### cultivation instructions

1. install [Processing](https://processing.org/) (version 4.0+).
2. clone this repository into your local soil.
3. open any sketch folder (e.g., `nodule_001/`) and run the `.pde` file.
4. let the code grow. adjust parameters. observe mutations.
5. the resulting images belong to the network. share them if the underground permits.
